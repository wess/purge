macOS · single-file binary · zero leftovers

Drag-to-trash leaves a mess. prg doesn't.

purge (prg) is a simple command-line uninstaller for macOS apps. It removes the app bundle and every file it left behind — preferences, caches, support files, and receipts — by parsing the bundle's metadata and scanning the directories macOS actually uses.

Features

An uninstaller that actually finds the leftovers.

No GUI, no kernel extensions, no daemon — just a fast binary that reads the bundle and checks the places macOS hides files.

📦

Bundle metadata parsing

Extracts CFBundleIdentifier and CFBundleName straight from the app's Info.plist to know what to look for.

🔎

Targeted scan paths

Searches only standard macOS locations — ~/Library/Preferences, ~/Library/Caches, /var/db/receipts, and more — for speed and accuracy.

☑️

Interactive selection

Presents a checklist of every candidate file so you choose exactly what to keep and what to delete.

👀

Dry-run preview

Preview all candidate files with --dry-run before anything is trashed. (Coming soon.)

⚡

Single-file binary

Shipped as a self-contained executable via bun build --compile — no Bun runtime required on the target machine.

🖥️

Built for macOS

Knows where Apple apps stash preferences, caches, support files, and package receipts — so you don't have to.

How it works

Point it at a .app and confirm.

Four quick steps from one command to a clean system — with you in control the whole way.

Read the bundle

prg parses the app's Info.plist for its bundle identifier and name.

Scan known paths

It checks the standard macOS directories where related files live, matching on identifier and name.

You choose

An interactive checklist lets you uncheck anything you want to keep.

Trash the rest

Press Enter and the remaining files — plus the app bundle — go to the Trash.

Get started

Installed in a minute.

Install with Homebrew, then point prg at any app bundle.

terminal 
# install with Homebrew
brew tap wess/packages
brew install purge

# uninstall an app and everything it left behind
prg /Applications/SomeApp.app

# preview without deleting anything
prg --dry-run /Applications/SomeApp.app

Prefer to build it yourself?

build from source 
# clone the repo
git clone https://github.com/wess/purge.git
cd purge

# install deps and build (macOS x64 + arm64)
npm install
npm run build:mac

# link the binary onto your PATH
ln -sf "$(pwd)/dist/prg-macos-$(uname -m)" /usr/local/bin/prg

Uninstall like you mean it.
